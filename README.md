# Piranhabon

convert **your** NAIST timetable into Google Calendar

## Usage

1. Create a `.env` with `.env.sample` as a reference.
2. Run script

```sh
$ poetry install
$ poetry run python piranhabon.py
```

### or Deploying to Heroku

```sh
$ heroku create
$ git push heroku main
$ heroku open
```
